This is a custom module for Drupal 8 created by Lil Engine.

The purpose of this module is to replace back ticks used in text boxes with code wrappers so it can inherent your sites code format, similar to how back ticks are used on Slack.

More details about this module and the motivation behind it can be found in this article:
https://www.lilengine.co/articles/syntax-highlighting-and-code-filters
